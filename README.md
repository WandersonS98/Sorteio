# Sorteio

O **Sorteio** é um programa que ajuda a sortear uma pessoa com uma lista fixa de alguns participantes e com opção para adicionar mais participantes.

## Tecnologias Utilizadas

- JavaScript
- HTML
- CSS

## Como Usar para sortear

1. Clone este repositório para sua máquina local:

2. Abra o arquivo `index.html` em um navegador.

3. Clique no botão para sortear um nome.

4. O programa sorteará um nome da lista aleatóriamente.

5. Para adicionar um participante a lista digite o nome do participante e clique em "Adicionar", o novo nome aparecerá na lista e poderá ser
sorteado. 


## Exemplo de Uso

- **Clique em sortear para sortear um nome da lista aleatóriamente
- **Insira um novo nome e clique em "Adicionar" para adicionar um novo nome na lista

**Resultado:**
- Exibirá um vencedor do sorteio
- Adicionará um novo nome na lista dos participantes
